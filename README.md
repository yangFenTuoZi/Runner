### 功能

- [x] 命令执行实时输出
- [x] 分离终端扩展包
- [x] 终端bash, busybox
- [x] 允许进程后台运行
- [x] 允许执行前降低权限（可能仅 Root 可用）
- [x] 支持强杀进程
- [x] 支持杀所有子进程
- [x] 增加进程管理
- [x] 支持自定义环境变量
- [ ] ~~开机自启动执行命令~~

- [x] App数据备份与还原
- [x] 终端
- [ ] 终端侧载模块（类似 dpkg ）
- [ ] 外部CLI
